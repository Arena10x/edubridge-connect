## EDUBRIDGE-CONNECT Server

### 1) Create database
```sql
CREATE DATABASE edubridge_connect;
```

### 2) Create table
Run the SQL in `db/init.sql`.

### 3) Configure environment
Copy `.env.example` to `.env` and update values if needed.

### 4) Install and run
```bash
npm install
npm run dev
```

Server runs at `http://localhost:5000`.
